# HelmCode Portal

Full-Loop AI Coding Platform — 业务场景 × 功能点全景视图(MVP Goal 1)。

## 快速启动

```bash
cd apps/portal
pnpm install
pnpm dev
```

打开 <http://localhost:3000> 看 5 域 40 功能点全景。

## 当前能力(Goal 1)

- 首页:全景矩阵(5 域 × 40 功能点 grid + 7 状态图例)
- 数据源:`data/feature-matrix.yaml`(mycmdeliverhub)
- 点击 feature 卡片(详情页由 Goal 2 实现)

## 技术栈

Next.js 15 (App Router) · React 19 · Tailwind 3.4 · TypeScript strict
